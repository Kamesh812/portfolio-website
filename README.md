# Web Developers - Business Website

Professional website for **Web Developers**, a Chennai-based web design agency.

## Open locally

Double-click `index.html` or run:

```bash
npx serve .
```

Then open http://localhost:3000 in your browser.

## Project structure

- `index.html` - Main page (all sections)
- `css/style.css` - Main styles
- `css/responsive.css` - Mobile and tablet layouts
- `js/main.js` - Menu, animations, contact form
- `assets/images/` - Add portfolio screenshots here

1. **Portfolio** - Replace sample projects with real client work in `index.html`

4. **Social links** - Add Instagram/LinkedIn in footer when ready

## Deploy free

- [Netlify](https://netlify.com) - Drag and drop this folder
- [GitHub Pages](https://pages.github.com) - Push to a repo and enable Pages
- [Vercel](https://vercel.com) - Import project folder
